# LAZO – Tienda de Carteras y Accesorios

Landing page moderna para una tienda de carteras, mochilas y accesorios, enfocada en una experiencia simple, visual y orientada a la conversión mediante WhatsApp.

---

## Demo
https://kevinflores198.github.io/lazo

---

## Descripción

**LAZO** es una tienda online que ofrece carteras y accesorios diseñados para el uso diario.  
El objetivo de esta landing es presentar los productos de forma clara, generar confianza y facilitar el contacto directo con el cliente.

La experiencia está optimizada principalmente para dispositivos móviles, donde se concentra la mayor parte del tráfico.

---

## Objetivos del proyecto

- Crear una landing page atractiva y moderna  
- Usar Tailwind CSS en un proyecto real  
- Mejorar habilidades en UI/UX  
- Generar conversión directa vía WhatsApp  

---

## Tecnologías utilizadas

- HTML5  
- Tailwind CSS  
- JavaScript  
- Swiper.js (carousel de productos)  
- Flowbite (componentes UI)  

---

## Diseño

- Paleta de colores cálidos (beige, marrón, blanco)  
- Color de acento: naranja/rosa fuerte para destacar acciones  
- Tipografía simple y legible  
- Enfoque visual en productos (imágenes optimizadas)

## Proceso del proyecto

El desarrollo de LAZO se realizó en distintas etapas, simulando un flujo de trabajo real con cliente:

### Primera reunión
- Definición de paleta de colores (tonos cálidos: beige, marrón, blanco)
- Elección de estilo tipográfico simple y moderno
- Recepción de imágenes reales de productos para utilizar en la web

---

### Segunda reunión
- Presentación del diseño en Figma
- Aprobación general del cliente
- Ajustes solicitados en colores y detalles visuales

---

### Tercera reunión
- Desarrollo completo del layout en HTML + Tailwind
- Estructura principal finalizada
- Pausa temporal del proyecto

---

### Etapa final
- Retoma del proyecto
- Optimización visual y responsive
- Preparación para publicación
- Integración con dominio
